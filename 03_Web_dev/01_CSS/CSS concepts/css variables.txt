css variables are used to declare a style once then use it throughout our entire website

// declaring variables
root refers to html tags
variables names always begain with --

--variable_name:value;
:root {
      --mainColor: orange;
      --mainFont: arial, sans-serif;
      --boxShadow: 12px 5px 10px grey;
}



// using variables
property: var(--variable name)
#logo{
  color: var(--mainColor);
}

1. CSS variables are  case sensitive

:root {
      --mainFont: arial, sans-serif;
}

// this wont work
font-family: var(--mainfont);

2. you can use css variables with other variables too

--newColor: red;
--newBorder: 2px solid var(--newColor)

border: var(--newBorder);

3. you can also write variables inline css

<h2 style = "border: var(--newBorder)">Text</h2>

4. while using css variables you can also give fallback values in case
you made a typo or if the browser is not able to render the specified css

--newColor: red;
color: var(--newColor, blue);

5. you can also define css variables for a particular element or scope

#scoped{
   --scopedFontSize: 30px;
}

#scoped button{
    font-size: var(--scopedFontSize);
